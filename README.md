# FullSearch

## 動機
這是實驗室新生訓練的內容，藉由實作簡單的動態影像編解碼，了解當中的一些基礎影像壓縮編碼技術。

## 描述
輸入一個 QCIF 檔，以 Motion estimation 、 Motion compensation 的方式，使用 Full Search 演算法進行區塊比對，最後輸出重建畫面、Motion Vector、Difference、Residual，以及每張重建畫面與原始畫面的 PSNR、SSIM 評估指標。

## 啟動方法
- 請確保你的 C++ 支援 CLI/CLR
- 直接建置執行
